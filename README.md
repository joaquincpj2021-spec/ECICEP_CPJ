# Portal ECICEP CPJ V3

Sitio web multipágina para GitHub Pages.

## Cómo subirlo

1. Descomprime este ZIP.
2. En tu repositorio GitHub, sube todos los archivos y carpetas:
   - index.html
   - asistente.html
   - mapa.html
   - ingreso.html
   - control.html
   - seguimiento.html
   - domicilio.html
   - gestion.html
   - registro.html
   - some.html
   - herramientas.html
   - biblioteca.html
   - css/
   - js/
   - docs/
3. Mantén GitHub Pages en branch `main` y carpeta `/root`.
4. Abre el link publicado.

## Cómo agregar documentos

Sube PDFs o DOCX a la carpeta `docs/` y luego edita `biblioteca.html` para agregar enlaces tipo:

```html
<a href="docs/protocolo-msf05.pdf">Ver protocolo</a>
```
